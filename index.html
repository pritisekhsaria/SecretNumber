Guess The Secret Number

Game objective:

  • Guess the secret number between 1 - 100, randomly generated by the computer

  • User gets 5 tries to enter the correct number

  • Computer will display an appropiate message, depending on whether the secret number is greater or less then the number you've entered.

  • A difference of 5 numbers or less, will display the message: "You are so close!!!"

  • The list of previously entered numbers is displayed at bottom of the screen

  • Hint: Always start with the mid point of your range
